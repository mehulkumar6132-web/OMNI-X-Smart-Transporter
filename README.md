# 🤖 OMNI-X: Operational Multi-Network Intelligent Transporter

**OMNI-X** is an IoT-enabled smart delivery robot prototype designed for indoor logistics. It uses the **ESP32** microcontroller and **Google Firebase** to achieve global remote navigation through a real-time web dashboard.

---

## 🌐 Project Overview
* **Connectivity:** IoT-based control via WiFi (Unlimited Range).
* **Safety:** Real-time obstacle avoidance using Ultrasonic sensors.
* **Interface:** Visual feedback through a 0.96" OLED display.
* **Backend:** Google Firebase Real-time Database.

## 🛠️ How to Use
1. Upload the code from the `Firmware` folder to your ESP32.
2. Connect your hardware as shown in the `Hardware` section.
3. Open the live dashboard link to control the robot remotely.
