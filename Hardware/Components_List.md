# Bill of Materials (BOM) - OMNI-X Prototype

The following components were used to build the OMNI-X Smart Transporter. This list ensures the modularity and replicability of the hardware.

| S.No | Component Name         | Quantity  | Key Function |
| :--- | :--- | :--- | :--- |
| 1 | ESP32 DevKit V1           | 1 |       Main microcontroller with built-in WiFi for IoT connectivity. |
| 2 | L298N Motor Driver        | 1 |       Controls the speed and direction of the DC motors. |
| 3 | DC Gear Motors            | 4 |       Provides locomotion to the 4WD chassis. |
| 4 | HC-SR04 Ultrasonic Sensor | 1 |       Measures distance for obstacle avoidance and safety. |
| 5 | SSD1306 OLED Display      | 1 |       0.96-inch screen for real-time status and HMI feedback. |
| 6 | SG90 Micro Servo          | 1 |       Rotates the ultrasonic sensor for environmental scanning. |
| 7 | 18650 Li-ion Batteries    | 2 |       7.4V power source for the entire system. |
| 8 | Piezo Buzzer              | 1 |       Provides acoustic alerts for startup and obstacles. |

---
**Note:** All interconnections are made using high-quality jumper wires and a centralized power switch.
